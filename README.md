# scratchmemo
# 概要
xyzzyのscratchバッファをメモとして活用し、
終了時に自動で内容を保存する仕組みを実装した。
# 背景
xyzzyで計算やメモを行う際、
毎回ファイル名を付けて保存するのが手間だった。
# 解決方法
.xyzzyにコードを追加し、
エディタ終了時にscratchの内容を任意フォルダへ自動保存するようにした。
# 効果
- 手動保存が不要になり作業効率が向上
- 一時メモを気軽に残せるようになった
- 計算ログの蓄積が容易になった
# 使い方
- .xyzzyに本コードを追加
- xyzzyを通常通り使用
- 終了時に自動でファイル生成

※scratchバッファの誤削除防止のため、デフォルトではkillできない設定にしています。必要に応じて無効化可能です。
